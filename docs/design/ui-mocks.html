How the shortlist composes

D1 + D2 coexist (left-click popover, right-click menu) and D3 is D1 with a pin — so "pick one" is really "build D1, get all three." On editing: start with E1 cards hosting the canonical peer form; ship E5 as the upgraded Ctrl+E raw mode (same parser, add the lint gutter); keep E3 in the drawer as the natural evolution of E1 if peer counts grow — the form is identical, only a rail is added. E7 is the no-new-windows fallback if the settings-window split ever feels wrong in practice.

The three form rules — stock-first ordering, explicit Metric with derived rank, ping-gated health knobs — apply identically in all four editors, so this decision can change later without relearning anything.
